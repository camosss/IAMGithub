# IAMGithub

RxSwift와 MVVM를 사용한 Github Repository Star 검색 클라이언트 애플리케이션

<br>

## Configuration

- OAuth
- User Infomation
- User Repository
- User Starred Repository
- Search Repository

<br>

## Commit Message Rule

- feature: 새로운 기능 추가
- fix: 버그 수정
- docs: 문서 관련
- refactor: 코드 리팩토링
- test: 테스트 코드
- chore: 빌드 업무 수정, 파일 수정 등..
- add: 추가
- edit: 수정
- delete: 삭제
- rename: 이름 변경
- correct: 문법 오류, 타입 변경, 오타 등..
